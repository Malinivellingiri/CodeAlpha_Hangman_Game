                                              # Hanuman Hangman Game
This is a Python-based Hangman game with a fun twist inspired by Lord Hanuman.
The game was created as part of the **CodeAlpha Python Programming Internship** to demonstrate Python skills and logic building.
## About the Game
- The player is asked to guess the hidden word one letter at a time.
- The word is related to Hanuman or Indian mythology.
- The game checks if the input is an alphabet and prevents wrong inputs.
- The player wins if they guess the word correctly before running out of chances.
- If the player fails, the game ends with a “Try again” message.
## Features
- Console-based Python game
- Hanuman-themed word selection
- Alphabet input check
- Win/Lose logic
- Clean and beginner-friendly code
